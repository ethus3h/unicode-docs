Unicode 2.1.0

Version 2.1.0 has been superseded by the latest version of the Unicode Standard.

Version 2.1.0 of the Unicode Standard consists of Unicode Technical Report #8, The Unicode Standard, Version 2.1 and the Unicode Character Database (UCD).

No contributory data files were released specifically for Version 2.1.0. Instead, there were four separate update versions, each of which involved the release of updated contributory data files. Links for each of these are provided below, for convenience in reference.

A complete specification of the contributory files for Unicode 2.1.2 is found on the page Components for 2.1.2.

A complete specification of the contributory files for Unicode 2.1.5 is found on the page Components for 2.1.5.

A complete specification of the contributory files for Unicode 2.1.8 is found on the page Components for 2.1.8.

A complete specification of the contributory files for Unicode 2.1.9 is found on the page Components for 2.1.9.
