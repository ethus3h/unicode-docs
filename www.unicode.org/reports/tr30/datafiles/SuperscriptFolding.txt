#
#	Map all characters that are superscripted forms to their equivalent letter
#
#	Format
#		Field 0 - character code (source)
#		Field 1 - character code of code sequnece (target)
#		Comment - utf8 of source -> utf8 of target, followed by
#				  characer name of source -> character name of target
#
#   Notes:
#
#	1) This is a superset of characters with the compatibility
#	decomposition <super>
#
#	Stability
#		The repertoire of modifier letters is not yet complete, therefore
#		this file would need to change whenever additional modifier 
#		letters are added to the Unicode Standard.
#
