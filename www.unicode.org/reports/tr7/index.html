Superseded Unicode Technical Report
 

The material formerly covered by UTR #7: Plane 14 Characters for Language Tags has been incorporated into the Core Specification of the Unicode Standard, starting with Version 3.1, and that UTR is thus superseded.

The text was incorporated into Version 3.1.0 in the following section:
  
        13.7 Tag Characters

For the most up-to-date version of this information, see the latest version of the Unicode Standard.
