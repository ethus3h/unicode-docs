Unicode Technical Note #11

Representing Myanmar in Unicode:
Details and Examples

Version 4
Authors Martin Hosken
Date 2012-12-13
This Version http://www.unicode.org/notes/tn11/tn11-4.html
Previous Version http://www.unicode.org/notes/tn11/tn11-3.html
Latest Version http://www.unicode.org/notes/tn11/


Summary

This document provides an introduction and overview on how to encode Myanmar language text. It addresses some of the outstanding issues in Myanmar encoding and provides many examples of words, their rendering and encoding. It also gives information on other languages using the script.
