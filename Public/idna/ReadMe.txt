http://www.unicode.org/Public/idna/

This public directory contains data files for UTS #46: Unicode IDNA Compatibility Processing.

The directories contain data files associated with a given version of Unicode, such as:

    http://www.unicode.org/Public/idna/6.0.0

The data files for the latest approved version are also in the directory:

    directory http://www.unicode.org/Public/idna/latest 
    
For more information, including the status of this data, see http://www.unicode.org/reports/tr46/.
