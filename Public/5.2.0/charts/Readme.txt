The Unicode Standard, Version 5.2 - Code Charts

The file "CodeCharts-noHan.pdf" contains all charts for Unicode 5.2
except for Han script blocks that are printed in multi-column format.

The file "CodeCharts-MulticolHan.pdf" contains the multi-column
Han blocks.

These two large code chart files are not independently paginated.
Instead, they are paginated as if they were assembled as a single
large file of 2000+ pages, in Unicode code point order.

The file "RSIndex.pdf" is the complete Unihan Radical
Stroke Index.

These files are stable. They will not be updated with errata, or
when additional characters are assigned to the Unicode Standard. See
http://www.unicode.org/charts/ for access to a complete list of the
latest character code charts. See http://www.unicode.org/charts/PDF/Unicode-5.2/
for charts showing only the characters added in Unicode 5.2. Both of
these locations provide access to multiple files for convenient on-line
viewing of individual code tables.

Fonts
The shapes of the reference glyphs used in these code charts are not
prescriptive. Considerable variation is to be expected in actual fonts.
The particular fonts used in these charts were provided to the Unicode
Consortium by a number of different font designers, who own the rights
to the fonts. See http://www.unicode.org/charts/fonts.html for a list.
