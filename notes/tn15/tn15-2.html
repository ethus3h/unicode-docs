Unicode Technical Note #15

Text conversion From
TSCII 1.7 to Unicode

Version 1.1
Authors Muthu Nedumaran

Previous Version http://www.unicode.org/notes/tn15/tn15-1.html

Summary

This document is written to assist with the conversion of TSCII encoded Tamil text to Unicode.

TSCII is an 8bit glyph encoding scheme used to exchange and store electronic text in the Tamil language prior to the emergence of Unicode enabled platforms that support Tamil Unicode.

Version 1.1 : Updated to reflect new characters introduced in Unicode 4.1
